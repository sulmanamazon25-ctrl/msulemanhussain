# Suleman Hussain — Founder World

Living founder operating system for [msulemanhussain.com](https://msulemanhussain.com).

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Structure

- Homepage narrative: Entry → Now → Product World → How I Think → Advantage → Archive → Build Log → Next
- Nav: NOW · PRODUCTS · BUILD LOG · EXPERTISE · ABOUT · LET'S BUILD
- Content: `src/content/`

## Color — Signal Foundry

Ink / bone / signal / phosphor / cobalt / amber / orchid — interactive accents, not generic purple templates.
